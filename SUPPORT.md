# Support

## Getting Help

For usage questions and troubleshooting:

1. Read `README.md`.
2. Read `docs/README.md`.
3. Open a GitHub Issue with reproduction steps and logs.

## Bug Reports

Use GitHub Issues and include:

- environment details
- exact commands or payloads
- expected vs actual behavior
- relevant logs/artifacts

## Security Issues

Do not open public issues for security vulnerabilities.
Follow `SECURITY.md`.
